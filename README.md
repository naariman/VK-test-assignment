# Симулятор распространения вируса
iOS-приложение, которое визуализирует распространение инфекции в группе людей. <br/> 
Сделано, как тестовое задание в VK. <br/> 

## Реализовано:
1) Алгоритм подсчета заражения, не в UI - потоке
2) Юзер френдли UI
